* Description
A command line that convert excel to html

* Installation
#+BEGIN_SRC bash
git clone https://github.com/alienzj/excel-preview
cd excel-preview
cargo build
#+END_SRC

* Run
#+BEGIN_SRC bash
# eg: data/xxx.xlsx
cargo run --bin excel-preview <path/to/excel> > hello.html
firefox hello.html
#+END_SRC
